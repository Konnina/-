• Описание

Приложение при запуске открывает текущую рабочую директорию и выводит список файлов и папок. Можно перемещаться по списку с помощью клавиш-стрелок, открывать папки, возвращаться в родительскую директорию, удалять элементы, отмечать файл или папку, а затем копировать или перемещать отмеченный элемент в текущую директорию.

• Возможности

- просмотр содержимого текущей директории;
- перемещение курсора по списку файлов и папок;
- открытие выбранной директории;
- переход в родительскую директорию;
- удаление выбранного файла или директории;
- отметка файла или директории;
- копирование отмеченного элемента в текущую директорию;
- перемещение отмеченного элемента в текущую директорию;
- просмотр содержимого текстового файла;

• Управление

| Клавиша | Действие |
|---|---|
| Стрелка вверх | Переместить курсор вверх |
| Стрелка вниз | Переместить курсор вниз |
| Enter | Открыть выбранную директорию |
| Backspace | Перейти в родительскую директорию |
| Delete | Удалить выбранный файл или директорию |
| Space | Отметить выбранный файл или директорию |
| F1 | Скопировать отмеченный элемент в текущую директорию |
| F2 | Переместить отмеченный элемент в текущую директорию |
| F3 | Просмотреть содержимое выбранного файла |
| Esc | Выйти из программы |

• Требования

- Операционная система: Windows;
- Компилятор с поддержкой C++17;
- Поддержка стандартной библиотеки `<filesystem>`.

Проект использует Windows-зависимые заголовочные файлы:

```cpp
#include <conio.h>
#include <windows.h>
#include <io.h>
#include <fcntl.h>
```

Поэтому программа рассчитана на запуск в Windows-консоли.

• Сборка

• Visual Studio

1. Создайте новый консольный проект C++.
2. Добавьте файл `FileManager2.cpp` в проект.
3. Установите стандарт языка C++17 или новее.
4. Соберите и запустите проект.

Также можно собрать через Developer Command Prompt:

```bash
cl /EHsc /std:c++17 FileManager2.cpp
```

• MinGW / g++

```bash
g++ -std=c++17 FileManager2.cpp -o FileManager.exe
```

Запуск:

```bash
FileManager.exe
```

• Структура кода

Основные части программы:

- `struct Item` — хранит путь к файлу или директории и признак директории;
- `load()` — загружает список элементов текущей директории;
- `draw()` — выводит интерфейс файлового менеджера;
- `key()` — обрабатывает нажатия клавиш;
- `openDir()` — открывает выбранную директорию;
- `goParent()` — выполняет переход в родительскую директорию;
- `del()` — удаляет выбранный файл или папку;
- `mark()` — запоминает выбранный элемент;
- `copyMarked()` — копирует отмеченный элемент;
- `moveMarked()` — перемещает отмеченный элемент;
- `viewFile()` — выводит содержимое выбранного файла;
- `main()` — основной цикл программы.

• Примечания

Операции удаления и перемещения выполняются сразу после нажатия соответствующей клавиши. Перед тестированием рекомендуется запускать программу в отдельной папке с копиями файлов.
